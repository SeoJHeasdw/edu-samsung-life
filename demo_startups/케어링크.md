# 케어링크 (CareLink)

- **설립** 2021년 · 서울 성동구
- **인원** 42명
- **한 줄 소개** 만성질환자 복약·검진 일정을 관리하는 모바일 건강관리 앱

## 사업모델
B2C 구독(월 9,900원) + 제약사 대상 복약 순응도 데이터 리포트 판매(B2B).
현재 매출 비중은 구독 65% / 데이터 35%.

## 규모
- 누적 가입자 38만 명, 월간 활성 사용자 11만 명
- 2025년 매출 62억 원, 영업이익 -14억 원
- 시리즈 B 200억 원 유치 (2025년 3월)

## 강점
만성질환자 리텐션이 12개월 68%로 카테고리 평균(41%)을 크게 상회.

## 리스크
개인 건강정보 활용 범위가 마이데이터 고시 개정에 직접 영향을 받음.
제약사 매출이 상위 2개사에 집중(78%).
