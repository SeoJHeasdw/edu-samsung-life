## 데모 진행

**Desktop** — 파일 3개만 끌어다 넣고 대화로 비교 → "즉각적이지만, 8개라면? 30개라면?"

**Code** — 이 폴더를 통째로 주고:

> "이 폴더의 회사 소개서를 전부 읽고, 생명보험사 관점에서 제휴 후보를 비교하는 표를 만들어 `비교표.md`로 저장해줘. 사업모델 · 규모 · 규제 리스크 · 우리와의 시너지 열을 넣고, 마지막에 상위 3곳과 그 이유를 적어줘."

→ **파일이 생겨나는 순간**을 보여주는 것이 이 데모의 목적입니다.
