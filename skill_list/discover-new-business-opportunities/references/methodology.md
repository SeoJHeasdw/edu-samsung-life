# 신사업 발굴 방법론

## 분석 단위

- **섹터:** 동일한 수요·규제·기술 신호를 공유하는 시장 영역
- **밸류체인 단계:** 투입물, 핵심기술, 생산, 유통, 서비스, 고객, 회수·재활용
- **비즈니스 모델:** 특정 고객과 지불자가 있고 수익모델을 설명할 수 있는 사업 단위

기술명, 제품 기능, 기업명을 비즈니스 모델과 혼동하지 않는다.

## 조사 우선순위

1. 정부 통계, 규제기관, 기업 공시
2. 산업협회, 표준기관, 학술자료
3. 기업 공식 자료와 제품 문서
4. 신뢰할 수 있는 언론과 유료 리서치
5. 검색 결과, 블로그, 홍보자료

하위 출처만 존재하면 신뢰도를 낮추고 공식 확인이 필요하다고 표시한다.

## 1–5점 루브릭

모든 점수는 정수 1–5로 입력한다. 점수마다 최소 한 개의 근거 또는 명시적 내부 가정이 필요하다.

| 기준 | 1점 | 3점 | 5점 |
|---|---|---|---|
| 시장규모 | 틈새·제한적 | 의미 있는 인접시장 | 대규모·복수 고객군 |
| 시장성장 | 정체·감소 | 완만한 구조 성장 | 강한 구조 성장 |
| 마진잠재력 | 원가·가격압박 큼 | 평균적 | 반복매출·고마진 가능 |
| 도입긴급성 | 선택적 | 계획 예산 존재 | 규제·비용 문제로 즉시 필요 |
| 규제순풍 | 명확한 역풍 | 중립·혼재 | 정책·규제가 수요 촉진 |
| 성숙도창 | 너무 이르거나 늦음 | 실행 가능 | 진입 시점이 특히 유리 |
| 역량적합성 | 핵심 역량 부재 | 일부 보완 필요 | 기존 역량 직접 활용 |
| 고객접근성 | 신규 채널 필요 | 파트너로 접근 가능 | 기존 고객·채널 활용 |
| 실행가능성 | 자본·기간 부담 큼 | 관리 가능한 투자 | 빠른 MVP·확장 가능 |

## 점수 공식

```text
시장성 = 시장규모×0.35 + 시장성장×0.40 + 마진잠재력×0.25
적시성 = 도입긴급성×0.45 + 규제순풍×0.30 + 성숙도창×0.25
전략적합성 = 역량적합성×0.40 + 고객접근성×0.30 + 실행가능성×0.30

총점 = 시장성×0.40 + 적시성×0.25 + 전략적합성×0.35
```

기본 가중치는 사용자가 다른 기준을 승인하면 변경할 수 있다. 가중치 합은 반드시 1이어야 한다.

## 근거 신뢰도

- **높음:** 근거 4개 이상이고 평균 품질 4점 이상
- **중간:** 근거 2개 이상이고 평균 품질 3점 이상
- **낮음:** 그 외

신뢰도는 사업 매력도가 아니라 점수 근거의 충실도를 뜻한다. 낮은 신뢰도의 고득점 후보는 `추가 검증`으로 분류한다.

## HITLs

### Gate 1 — 조사 범위

- 지역, 기간, 제외 영역, 섹터 경계가 명확한가?
- 검색 결과 편향을 줄일 모집단과 출처가 정해졌는가?

### Gate 2 — 기준과 후보

- 평가기준이 회사 전략과 연결되는가?
- 가중치가 특정 결론을 만들기 위해 사후 조정되지 않았는가?
- BM 후보의 고객·지불자·수익모델이 명확한가?

### Gate 3 — 심층조사

- 핵심 수치가 같은 정의와 기간을 사용하는가?
- 반대 근거와 데이터 공백이 표시됐는가?
- 제휴·인수 후보를 거래 가능성이 확인된 것처럼 표현하지 않았는가?

### Gate 4 — 의사결정

- 추천과 근거 신뢰도를 분리했는가?
- 되돌릴 수 있는 다음 검증 행동이 있는가?
- 최종 투자·법률·회계 판단을 사람이 수행하는가?
